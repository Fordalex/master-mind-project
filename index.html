Master Mind

How To Play

Aim Of The Game

Mastermind is a game that requires a player to use logic, speculation and reasoning skills to determine the sequence of the coloured counters. The computer has secretly generated a sequence leaving players having 10 attempts to guess correctly, or the computer wins!.

Step: 1

Place your first 4 counters trying to workout what coloured counters the computer has generated. (You might want to start filling the first row with counters all of the same colour)

Step: 2

If you got any coloured counters right, the computer will return black and white pegs. (Black: Right colour, right place) (White: right colour, wrong place) or none if the colour is not in the hidden sequence.

Step: 3

Use a process of elimination to work out what is hidden under cover.
